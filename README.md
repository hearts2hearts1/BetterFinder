# BetterFinder
Bringing back the MLS course offerings experience.

## Installation
### 1. Download the latest release
[Download BetterFinder](https://github.com/hearts2hearts1/BetterFinder/releases/latest)

### 2. Extract the ZIP file
Extract it to a new folder on your computer.

### 3. Open your browser's extensions page
Go to `chrome://extensions/`

### 4. Enable Developer Mode
Toggle the switch in the top-right corner.

### 5. Load the extension
- Click **Load unpacked**
- Select the extracted BetterFinder folder

## Usage
Log in to your ArchersHub account, click the BetterFinder extension icon in your browser toolbar, and start searching for courses.

## Privacy
This extension runs entirely in your browser. No data is collected, stored, or sent anywhere. Your session and credentials stay yours.
